2026. 03 - 2026. 05 (개인 프로젝트)

LAUNCH WEBSITE

OVERVIEW

디자인의 비주얼 요소와 실제 퍼블리싱의 작동 마무리를 정교하게 정돈한 웹 아카이빙 프로젝트입니다. 타겟 유저에게 감성적이면서도 흐트러짐 없는 인터랙션을 전달하는 것을 핵심 목표로 잡고 빌드업했습니다.

UI/UX DESIGN PROCESS

image_b4c149.jpg 오브제 무드에서 영감을 받아 부드러운 우유빛 분홍색 그라데이션과 반투명 아크릴 느낌의 입체 글래스모피즘 시스템을 차용하여 설계했습니다.

UI Design System & Wireframe Zone

PUBLISHING POINT

하드코딩 인라인 방식을 탈피하고 Flex 및 Grid 시스템을 복합적으로 배치하여 화면 해상도 변화에 능동적으로 반응하는 모듈형 CSS 뼈대를 구축했습니다. 스크롤 인터랙션 시 부드러운 가속도를 제공합니다.
